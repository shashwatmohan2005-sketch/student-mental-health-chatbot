# Student Mental Health Chatbot

An AI-powered chatbot designed to analyze student messages and identify potential mental health risks using NLP and Machine Learning.

## Features

- Interactive chatbot interface
- Student-focused conversations
- Mental health risk detection
- NLP-based text analysis
- Streamlit frontend

## Tech Stack

- Python
- Streamlit
- Scikit-Learn
- NLP
- GitHub

## Team Members

- NLP Processing
- Machine Learning Model
